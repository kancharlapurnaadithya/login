# 🔐 SecureApp - Google Authentication System

## 📌 Project Description

SecureApp is a web-based authentication application developed using Firebase Authentication and Google Sign-In. The application provides a secure and user-friendly login experience by allowing users to authenticate using their Google accounts.

After successful authentication, users are redirected to a personalized dashboard where their profile information is displayed. The application ensures secure access management and prevents unauthorized users from accessing protected resources.

## 🔄 Authentication Flow

1. User opens SecureApp.
2. Clicks "Sign in with Google".
3. Firebase Authentication initiates Google Sign-In.
4. User selects Google account.
5. Authentication is verified.
6. Dashboard page loads.
7. User details are displayed.
8. User can logout securely.
